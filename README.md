# St. Lawrence S.S Ssonde — Prototype Site

This is a lightweight, responsive static prototype to pitch a website for St. Lawrence S.S Ssonde (Namugongo).

Quick start

1. Open `index.html` in a browser to preview locally.
2. Replace placeholder images, the YouTube `VIDEO_ID`, and the Google Maps embed with real assets.

What I scaffolded

- `index.html` — single-page prototype with sections: Home, About, Academics, Student Life, Admissions, Contact
- `css/style.css` — mobile-first responsive styles
- `js/script.js` — small helper for mobile nav and header interactions
